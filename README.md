# LineHeroWeb

「LineHero」LINE 遊戲的官方文字冒險遊戲宣傳網站。

## 簡介

LineHeroWeb 是 LINE 遊戲「LineHero」的官方網站，提供遊戲資訊、活動公告、資料等內容。讓玩家能更方便地參與 LineHero 的冒險旅程。
![LineHero 宣傳圖](src/assets/images/linehero_intro.png)

## 主要功能

- 遊戲資訊
- 遊玩方式
- 文字冒險遊戲互動
- 活動與獎勵資訊
